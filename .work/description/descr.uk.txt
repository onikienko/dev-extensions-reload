Розширення для розробників розширень.

Щоб перезавантажити активні розпаковані розширення використовуй клавіатурне скорочення (Alt-Z за замовчанням) або контекстне меню на будь-якій вкладці.

За:

- Швидке перезавантаження з контекстного меню на будь-якій вкладці або за допомогою клавіатури (за замовчанням Alt-Z, але можна змінити)
- Перезавантаження тільки активних (ввімкнених) розширень
- Якщо Options page відкрита, вона також буде перезавантажена
- Можливість також перезавантажити активну вкладку (див. сторінку налаштувань розширення). Корисно при розробці content script розширень
- Можливість відключити перезавантаження з контекстного меню (див. сторінку налаштувань розширення)

Проти:

- Таке перезавантаження закриває background debugger window (звичайно, якщо воно було відкрите). На даний момент нема публічного API для перезапуска.

Вихідний код - https://github.com/onikienko/dev-extensions-reload/
